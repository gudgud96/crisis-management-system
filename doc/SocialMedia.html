SocialMedia.controller module¶

class SocialMedia.controller.SocialMedia¶

Bases: object

SocialMedia is the main-entry point for the social media subsystem. It is a controller class that provides the following services:

  • Alert authorities about an incident
  • Alert public members residing near a crisis area about a crisis
  • Post to facebook with information about a crisis
